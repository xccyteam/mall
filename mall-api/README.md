# mall-api
## 准备工作
- node.js环境（npm包管理器）
- express-generator 脚手架构建工具
- cnpm  npm的淘宝镜像

### 搭建过程
- npm install -g express express-generator      全局安装
- mkdir api-demo
- cd api-demo
- npm init
- npm install express --save        安装express本地依赖
- express -e    


## 开发运行
```
    # 安装依赖
    npm install

    # 本地开发 开启服务
    npm start
```
浏览器访问 http://localhost:3100
